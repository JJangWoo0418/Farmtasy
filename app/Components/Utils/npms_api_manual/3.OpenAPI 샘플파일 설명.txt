1.병해충검색(사진검색)
  1.1 REST
      dgnss.jsp(asp,php)     : 사진검색 대분류 조회
      dgnss1.jsp(asp,php)    : 사진검색 중분류 조회
      dgnss2.jsp(asp,php)    : 사진검색 소분류(작물/잡초) 조회
      dgnss3_1.jsp(asp,php)  : 해충 상세조회
      dgnss3_2.jsp(asp,php)  : 곤충 상세조회
      dgnss3_3.jsp(asp,php)  : 천적곤충 상세조회
      dgnss3_4.jsp(asp,php)  : 병 상세조회
      dgnss3_5.jsp(asp,php)  : 잡초 상세조회
      XMLparse.php           : php용 xml 파서
      dgnssListTongHab.jsp   : 통합검색 목록 조회 - 검색조건 및 상세페이지 이동(dgnss3_1 ~ dgnss3_5)은 위의 예제 파일을 참고해서 작업하시기 바랍니다.
